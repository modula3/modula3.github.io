M3 Mailing Lists

Currently there are three mailing lists that are concerned with the development and use of Modula-3. These are

m3announce@elegosoft.com

This list is used for announcements concerning Modula-3, usually announcements of new releases and release notes. You can read an archive of old messages at

http://www.elegosoft.com/Zope/m3/m3announce
m3devel@elegosoft.com

This list is used discussions and questions regarding the development of Modula-3. You can read an archive of all messages at

http://www.elegosoft.com/Zope/m3/m3devel
m3commit@elegosoft.com

This list distributes the commit messages of the CM3 and PM3 CVS repositories. You can read all the commit logs since January 2003 at

http://www.elegosoft.com/Zope/m3/m3commit

Currently all the mailing lists are hosted at the Zope server of Elego Software Solutions, and are rather low-bandwidth. If you want to subscribe to any of these lists, send a mail to m3-support@elego.de.
