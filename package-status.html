CM3 5.1 Package Status

Abbreviations:
C = package compiles
U = package untested / status unknown
F = package produces a library or program that fails in one or more known ways
MI = package compilation breaks on missing or broken import
OK = package produces usable library or program
NU = package needs unicode / WIDECHAR update
NT = package needs text interface update (breaks on TextF)
