Installation Archives

These are the currently available minimal binary installation archives:

cm3-min-POSIX-FreeBSD3-5.1.0.tgz
cm3-min-POSIX-LINUXLIBC6-5.1.0.tgz
cm3-min-WIN32-NT386-5.1.0.tgz
cm3-min-POSIX-SOLgnu-5.1.0.tgz

You will need one of the binary archives to get a working compiler and one or more of the source distributions below. See the installation notes for further information.

Source Archives

cm3-src-all-5.1.0.tgz
This archive contains the complete source code. It is rather big (> 25MB).

cm3-src-sys-5.1.0.tgz
This archive contains all the system and tool source code as well as the standard libraries m3core and libm3. It is also rather big (> 21MB).

cm3-src-std-5.1.0.tgz
This archive contains a set of standard packages (libraries and tools) that are currently compilable and form a useful system including network communication and graphical user interface. It is not as big as the other archives (about 5 MB :-). The packages contained can be built with scripts/do-cm3-std.sh.

cm3-cvs-2001-02-19.tar.bz2
This is a snapshot of the complete CVS repository used to produce these releases. Note that it is compressed with bzip2, but is still very large. You do not need this unless you want to have a look at the development history or plan to setup your own CM3 project.

Updates

m3override-update.tgz
This file contains fixed m3overrides files containg complete overrides for all packages (except some test packages which have not yet been fixed) relative to ROOT.
